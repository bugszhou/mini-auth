### 返回码

| 返回码 | 说明 |
| --- | --- |
| 0 | 成功 |
| 5000 | 微信小程序code获取失败 |
| 5001 | 微信小程序发送请求超时 |
| 5002 | 微信小程序发送请求服务端无法连接 |
| 5003 | 微信小程序发送请求未知异常 |
| 5004 | 中间件重复调用 |
| 5005 | 中间件错误 |
